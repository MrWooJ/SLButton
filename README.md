# SLButton

Simple Loading Button
Very easy to use

# How to Use
import "SLButton.h"

Optional Properties :

+ AnimationDuration ( animation duration )
+ DisableWhileLoading ( disable button while loading )

# Methods Calling

[obj showloading] to loading mode

[obj hideLoading] to normal mode

# Requirements
+ iOS >= 7.0
